Food2Go

When travelling it isn't always easy to find ingredients and nutritional information about the local dishes. That's where Food2Go comes in. Food2Go is a single page application that allows a user to choose a country and see dishes from that country. When a specific dish is chosen, a list of ingredients and nutritional information is displayed.
